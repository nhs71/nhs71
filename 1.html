1. HBV DNA Real Time PCR

가. 검사법

Real Time PCR

  1. 검사코드 : LPD194
  2. EDI코드 : D7044030

참고치

나. 검사원리

HBV에 특이적인 sequence를 보유한 프라이머들은 Tag polymerase에 의해 증폭됩니다. 그와 동시에, 유전자에 특이적인 Tagman probe는 감소하면서 형광을 나타냅니다. 형광의 강도는 정량을 위해 실시간으로 real time PCR 검사 기기에 의해 측정됩니다. B형 간염바이러스 DNA를 정량하기 위하여, 정량표준곡선은 알려지고 정의된 표준농도를 사용하여 산출됩니다.real time PCR 검사 기기를 이용하여 측정되는 검체의 Ct값은 정량 표준 곡선에서의 Ct값과 비교되어, B형 간염 바이러스의 DNA농도가 산출됩니다.

※ 타겟 증폭 및 PCR산물의 검출의 원리 추출된 DNA검체는 2×PCR Mixture 각 tube(PCR tube)에 첨가됩니다. real time PCR검사기기는 두 가닥의 타겟 DNA의 변성을 위하여 반응 혼합액에 열을 가합니다. 혼합액의 온도가 내려감에 따라, 프라이머는 타겟 DNA에 결합(anneal)하게 됩니다. 마그네슘 이온(Mg²⁺)과 dNTP(dNTP, dATP, dCTP, dGTP, dUTP를 포함하여)가 존재하는 환경에서 열에 안정적인 Hot start Tag polymerase에 의해 amplicon이라 불리는 이중가닥의 DNA를 만들기 위해 타겟 DNA를 따라 결합된 프라이머가 연장됩니다. real time PCR검사기기는 amplicon DNA의 두 배만큼이 되도록 의도된 각 cycle을 가진 증폭프로그램에 따라, 자동으로 본 과정을 반복합니다. 이중 표지된 형광 프로브는 증폭과정동안에 방출되는 형광 염료의 방출강도를 모니터링함으로써 PCR산물의 축적을 실시간으로 검출 할 수 있도록 해줍니다. 개별적인 리포터 염료의 사용은 B형 간염바이러스 DNA와 내부대조액 DNA의 개별적인 구분이 가능하도록 해줍니다.signal의 Ct값은 PCR시작점에서 타겟DNA의 양과 관련이 있습니다.

다. 임상적의의

B형 간염 바이러스 (Hepatitis B Virus, 이하 HBV)는 바이러스성 간염 주요 원인 바이러스로 잘 알려져 있는데 HBV 간염을 진단하기 위해 흔히 HBV 표면 항원(이하 HBsAg)이나 HBVe antigen (이하 HBeAg)을 진단마커로 활용하여 혈액 내 존재유무를 판별하는 방법을 사용하고 있다. 하지만 바이러스의 생활사에 따라 조기진단을 하지 못하거나 HBV 감염환자임에도 혈액 내 HBsAg이나 HBeAg 등이 줄어들거나 존재하지 않아 HBV 치료를 받고 있는 환자에 대한 정확한 진단을 하지 못할 가능성이 있다. 이에 반해 HBV DNA를 검출하는 진단방법은 인터페론 알파와 라미부딘을 통한 HBV 치료를 받는 환자에게서 HBV DNA의 양이 치료기간에 따라 일정하게 줄어드는 양상을 관찰할 수 있어 치료에 따른 환자의 호전 상태 등을 파악할 수 있다. 또한 혈액 sample 내 HBV DNA를 검출하는 real-time PCR법은 면역혈청변환 이전에 HBV 감염을 보다 빠른 시기에 실시간으로 확인할 수 있다.

라. 검체조건

  1. 채혈용기 : SST tube
  2. 검체운송 : 전혈은 2~25℃에서 6시간 이내에 혈청분리
  3. 최소검체량: 혈청 300uL
  4. 검체보관
    1. 검사 전 : 혈청은 2~8℃에서 7일 이내 보관, 장기간 보관할 경우 -15℃이하(1개월 이내)에서 냉동보관
    2. 검사 후 : 혈청은 -15℃이하에서 1개월 냉동보관, 핵산추출물은 보관하지 않는다.
  5. 검체 냉동-해동에 따른 안정성 : 4회까지 냉동과 해동의 반복이 가능
  6. 간섭물질 : 헤파린

바. Calibration

  1. 시행주기

    1. 새로운 Box 사용 전
    2. Calibration 오류 시
    3. 시약 및 장비의 이상이 없는 상태에서 QC결과가 의심될 경우
    4. Lot 변경시

  2. 검사방법

    1. Kit내에 포함된 Standard1,2,3를 실온화 시킨다.
    2. 각각 2X PCR mixture 3곳에 10uL씩 분주하고 프라이머/프로브 5uL도 각각 넣는다.
    3. PCR을 진행한다(약 2시간 20분)

  3. 허용기준

    1. Slope : -3.70 ~ -3.00
    2. Intercept : 38.23~44.23
    3. standard 1(10^9) :HBV(10.4~12.4),HBV I.C(27.0~35.0)
    4. standard 2(10^6) :HBV(20.4~22.4),HBV I.C(27.0~35.0)
    5. standard 3(10^3) :HBV(30.4~32.4),HBV I.C(27.0~35.0)

  4. 계산방법 실제값(IU/mL) = [계산된 값(IU/mL) X 200 X y ] / [x X 50]

    1. 200 : 제시된 검체양, uL
    2. 50 : 제시된 최종 핵산 추출양, uL
    3. y : 실제 최종 핵산 추출양, uL
    4. x : 실제 검체양, uL

  1. 허용범위를 벗어난 경우

    1. 허용범위를 벗어난 경우 재검을 시행하고 허용범위 내에 들지 않을 경우 시약 및 검사진행 과정중의 문 제점을 확인한다. 새로운 Kit로 검사하여 허용 범위 내에 들면 내부정도관리 이상처리기록서에 기록하고, 허용범위를 벗어난 경우 UM과 전문의에게 보고하고 시약회사나 장비회사에 연락하여 조치를 취한다.

      • 정도관리 물질이 허용 범위 안에 들어올 때까지 환자 검사 결과는 보고하지 않는다.
      • 즉시 검사가 불가능한 환자검체는 검체 내 구성 물질이 변질되지 않도록 냉장 보관을 유지한다.
      • 모든 시약과 정도관리 물질의 유효기간을 확인한다.
      • Technique, reagents, calibration등의 검사 수행을 정확히 하였는지 확인한다.
      • 정도관리 분석을 다시 한다.
      • 만일 새 측정치가 범위 내 에 들면 결과를 QC 차트에 기록하고 환자검사를 보고한다 (첫 번 측정치는 random error-20회 검사 중 한 번의 확률의 가능성이 높다
      • 만일 새 측정치가 계속 상 하한치를 벗어나면 검사를 보류한다.
      • Calibration을 재시행한 후 정도관리를 재검사한다.
      • 만일 새 측정치가 여전히 상 하한치를 벗어나면 다음과 같은 원인을 예상하고 조치한다. 검사 장비의 이상 유무 ( 검사기기에 대한 적절한 Maintenance를 수행한다.) Standard 혹은 control의 이상 유무 검사시약의 이상 유무 ( 새로운 시약으로 교체 후 recalibration을 하고 Control 재검) 문제점의 원인을 분석한 내용과 조치사항, 처리결과, 검사자등을 내부정도관리이상처리 기록서에 기록하고 UM, 전문의에게 보고한다.

    2. 상기의 모든 사항에서 이상이 발견되면 즉시 조치를 취하고 기록을 남기며 이 모든과정을 적극적으로 관 리하며 책임진다. 다) 정도관리 오류대책

      • 검사 전 오류 : 정도관리 물질의 변질 여부 확인, Lot No 및 유효기간 확인, 시약 및 검체 보관조건 합당한지 확인
      • 검사 후 오류 : 정도관리 결과가 Mean±3SD를 벗어난 경우 계통오차와 우연오차의 발생을 감지하고 결과 보고를 보류한다.
      • Systematic error : 개인적인 습관, 측정기기의 계통적 부정확성, 시약의 불량, 측정환경의 계통적인 부적격 성, 측정법 자체의 습성
  2. 외부정도관리 : 운영지침서 참고
  3. 허용범위설정 방법
    1. 새로운 Lot No가 교체되기 전 5번 시행 후 Mean과 SD를 산정한다. SD는 이전에 사용한 결과값으로 대체하여 임시 사용 할 수 있다.
    2. 정도관리 물질의 누적자료가 20개가 되었을 때 Mean±3SD를 산정하고 전문의와 상의 후 검사실 허용 범위로 사용한다.
  4. AMR 검증
